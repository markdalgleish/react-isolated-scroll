# react-isolated-scroll

React component for [isolated-scroll](https://github.com/markdalgleish/isolated-scroll).

```bash
$ npm install --save react-isolated-scroll
```

## Usage

```js
import IsolatedScroll from 'react-isolated-scroll';

<IsolatedScroll>
  {/* children... */}
</IsolatedScroll>
```
